How Steganography Works

1. LSB (Least Significant Bit) Pixel Encoding

Every image is composed of pixels containing Red, Green, and Blue (RGB) color values ranging from 0 to 255. Our cipher converts your secret text into binary bits (0s and 1s) and replaces the very last bit of each pixel byte. Because altering the last bit changes color intensity by less than 1%, the difference is completely invisible to the human eye.

2. Why Document (.png) Format Prevents Data Loss

Lossy image formats like JPG / JPEG recompress and smooth out pixel colors to shrink file size. This compression alters the exact pixel values, destroying hidden LSB bits. Saving in Document (.png / lossless) format keeps every single pixel value 100% exact, guaranteeing zero data loss.

3. 100% Private Offline Processing

All encoding and decoding operations take place entirely inside your browser's HTML5 Canvas memory. No image or secret text is ever transmitted over the network or saved on remote servers.
